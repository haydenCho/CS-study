# CS 기술 면접 공부

인프라 직무(네트워크·클라우드·시스템·보안) 기술 면접 대비 자료 모음입니다.
전공자·정보처리기사 수준의 예상 질문과 모범 답안을 과목별·단원별로 정리했습니다.

- 총 **201문항** (네트워크 104 + 운영체제 97)
- 각 문항에 중요도 표기: ★★★ 최빈출 · ★★ 자주 출제 · ★ 기본 개념
- 모든 단원을 **HTML / PDF / Markdown** 3가지 형식으로 제공

## 시작하기

[`index.html`](index.html)을 브라우저로 열면 전체 과목 목록에서 시작할 수 있습니다.
과목별 목록(`index.html`)에서 단원별 문서로 이동하거나, 통합본(`*-all-in-one.html`) 한 파일로 전체를 볼 수 있습니다.

## 디렉토리 구조

```
CS/
├── index.html             # 메인 인덱스 (과목 선택 페이지)
├── network-interview/     # 네트워크 (8개 유닛 · 104문항)
│   ├── index.html                        # 단원 목록
│   ├── network-interview-all-in-one.html # 통합본
│   └── units/                            # 단원별 .html / .md / .pdf
└── os-interview/          # 운영체제 (8개 유닛 · 97문항)
    ├── index.html
    ├── os-interview-all-in-one.html
    └── units/
```

각 단원 문서는 질문 → 모범 답안 → 관련 개념 순으로 구성됩니다.

```markdown
## Q1. OSI 7계층을 각 계층의 역할과 함께 설명하세요. ★★★

**답안**
(면접에서 바로 말할 수 있는 수준의 모범 답안)

**관련 개념**
(꼬리 질문 대비용 연관 키워드)
```
